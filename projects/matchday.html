1. 프로젝트 개요

  • 프로젝트명 : JSP를 활용한 웹 구축
  • 프로젝트 구현내용
    • JSP 기반 Front-end 구현
    • 클라우드DB(AWS)기반 Back-end 구현
    • 웹소켓을 통해 실시간 채팅시스템 구현
    • 다양한 웹 기능 구현 (게시판, 채팅, 템플릿 등)
  • 프로젝트의 활용
    • 보드게임을 하고싶은 사람끼리 모일 수 있음
    • 평점관리를 통해 좋은 보드카페를 찾을 수 있음
    • 모르는 보드게임이나 하고싶은 보드게임을 질문 할 수 있음
    • 코드분리가 잘 되어있어 다른 곳에 쉽게 이식 가능

    • 2. 팀소개

    • 팀명 : Matchday
    • 팀원
