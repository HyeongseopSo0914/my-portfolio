1. 프로젝트 개요

  • 프로젝트명 : 공공데이터를 활용한 데이터 분석
  • 프로젝트 구현내용
    • Flask 기반 웹 애플리케이션 구현 및 Blueprint 구조 설계
    • MySQL 로컬 DB 연동을 통한 실시간 데이터 처리
    • Seaborn, Matplotlib, Scipy, Pandas로 통계 분석 (ANOVA, ANCOVA, 상관분석 등)
    • 다양한 웹 기능 구현 (대시보드, 사용자 인터랙션, 템플릿 등)
  • 프로젝트의 활용
    • 코로나 전후 5년간 데이터로 사회 변화 분석
    • 대중교통 이용 패턴 기반 운행 전략 수립
    • 인구 이동 트렌드 기반 도시계획 자료 제공
    • 기후 요인이 교통·인구에 미치는 영향 분석
    • 시각화 대시보드 기반 의사결정 지원

      • Golden Rule
      1. 회의는 수업시간에 주어진 프로젝트 시간에 하는걸 원칙으로 한다.
      2. 회의는 50분 진행 후 10분의 휴식 시간을 가진다.
      3. 회의의 마지막에는 다음 회의의 주제를 미리 정하고 마무리한다. 부득이하게 주제가 정해지지 않았더라도 회의는 그대로 진행하는 것을 원칙으로 한다.
      4. 서로의 의견은 항상 솔직하게 공유한다.
      5. 회의에 불참하게 된 사람은 매 회의 시작 시에 10분간 불참 사유에 대해 발표 시간을 가진다.
      6. 회의록은 담당자가 정리한다.
      7. 매 주 발견된 이슈나 논의 할 사항은 정해진 플랫폼을 통해 빠르게 공유하고 다음 회의 때 논의하도록 한다.

    7. 분석 결과

    항목 분석 결과 해석 요약
    지하철 이용량 ANOVA, Tukey HSD 유의함 (p < 0.001) 코로나 절정기에 급감, 이후 회복세
    버스 이용량 ANOVA, Tukey HSD 유의함 (p < 0.001) 지하철과 유사하게 변화, 절정기 감소 후 종식기부터 회복
    총생활인구수 ANOVA 유의함 (p < 0.001) 절정기 이전보다 감소, 이후 완만히 회복
    평균기온 ANOVA 유의하나 Tukey 미미 기온 변화는 계절성 영향이 큼
    일강수량 ANOVA 유의함 (p < 0.01) 시기별 분산 차이 존재, 절정기·종식기 이상치 영향
    ANCOVA (공변량: 지하철) 코로나 시기, 지하철 모두 유의함 (p < 0.001) 지하철 이용량이 버스 승객 수에 강한 영향
    설명력 (모델 한계) R² 유사 추정: 약 36.9% 전체 변화의 일부만 설명, 외부 요인 포함 필요

    결론: 코로나 시기에 따라 대중교통 이용량과 인구에 유의미한 변화가 있었으며, 특히 지하철과 버스는 절정기에 이용이 급감하고 이후 점진적으로 회복됨. 기후 변수는 계절적 요인의 영향을 더 크게 받는 것으로 해석됨. ANCOVA 분석을 통해 지하철 이용량이 버스 수요에 영향을 주는 것도 확인됨.
