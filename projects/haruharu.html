1. 프로젝트 개요

  • 프로젝트명 : NLU와 LLM을 이용한 챗봇개발
  • 프로젝트 구현내용
    • MySQL 로컬 DB연동을 통한 실시간 데이터 처리
    • 학습데이터(json)를 자체제작해서 모델학습 → Intent, Entity, LLM
    • Ollama를 활용한 LLM구현 → gemma:2b model을 활용해 prompt제약을 통한 응답구현
    • React.js를 활용한 웹앱 구현
  • 프로젝트의 활용
    • 사용자들이 원하는 응답을 해주는 봇 구현
    • 특정 주제에 대해 자세하고 유저친화적인 응답 구현
    • non-portable환경에서만 작업하던걸 portable한 환경에서 가능
    • 챗봇 플랫폼으로 향후 발전 가능성
    • 유저의 응답을 DB에 저장해 연속되는 대화가능성
